Help is good.

  • hangout: get a quick room for a hangout.
  • mkproject [projectname]: Create a project. letters and numbers only please
  • update [projectname] [status]
  • lsproject [nickname]: get all projects, defaults to you
  • wtf [nickname]: get latest updates for a single person
  • derp: get last 5 updates from each person
  • lsp [nick]: get projects for a user, defaults to self
  • rmproject [projectname]: remove a project
  • help: this help
